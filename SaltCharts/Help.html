Salt Charts Help
Navigating the Map

You can move around the map view in several ways. You can drag the map to the desired position directly, or drag the horizontal and vertical coordinate bars to move only one direction. You can also use the navigation buttons at the top left to move the map by half the screen width and center the map.

Creating a Waypoint

Create a waypoint by choosing any combination* of the island and marker buttons. Right click a cell on the map to place the waypoint. You can alter the waypoint appearance without removing names and notes by repeating the process over an existing waypoint.

*Deep Sea cannot have a marker. If a marker is selected with the Deep Sea type, it will be ignored.

Adding a Name and Notes

To name a waypoint or add notes, click an existing waypoint. In the resulting dialog you can type the name and notes. Names and notes will appear in the status bar at the bottom of the window when the mouse is moved over the waypoint.

Deleting a Waypoint

Delete a waypoint in the same way you would create one by selecting the No Island and No Marker buttons together. If the waypoint has a name or notes you will be asked to confirm the deletion.

Saving the Map

By default the map is automatically saved when a waypoint is created, deleted, altered, named, or has notes added. You can alter this behavior by clicking the settings button. When AutoSave is turned off a save button will appear in the toolbar.

Creating a New Map

Create a new map by clicking the New button in the toolbar. You will be asked to provide a name for the map, it is strongly suggested you use the seed name. Maps will always be created in the folder the application is located and will use the extension *.map

Loading a Map

Load an existing map by clicking the Open button in the toolbar.

Saving a Map as an Image

Save an image of the map by clicking the Save as Image button in the toolbar. You may save as several image formats including PNG, GIF, JPEG, BMP, and TIF.

About Salt Charts

For additional information about Salt Charts and links to online resources, click the About button in the toolbar.
